This is a Steam Minilleum plugin. If you haven't installed Minilleum you can type this command on your Windows Powershell: iwr -useb "https://steambrew.app/install.ps1" | iex   
This plugin allows you to add steam games in one click without a manifest lua generator. What you do after installing Luatools is open your Steam. Click on the "Steam" button. Minnileum - Plugins- Luatools- Save Changes.
